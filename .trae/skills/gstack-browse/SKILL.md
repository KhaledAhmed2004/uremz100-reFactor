# Gstack Browse Skill

Deep research ebong existing solutions khuje ber korar jonno ei skill use kora hoy. Trae er `WebSearch` ebong `WebFetch` tools use kore best engineering solutions khujte hobe.

---

## CRITICAL: Browsing Philosophy

1.  **Search Before Building**: Kono kichu implement korar age check koren "has someone solved this before?".
2.  **Layered Search**:
    - **Layer 1 (Tried and True)**: Standard libraries (e.g., Axios, Zod, Mongoose).
    - **Layer 2 (New and Popular)**: Modern but stable tools.
    - **Layer 3 (First Principles)**: Raw logic ebong custom implementation (only when Layer 1 & 2 fail).
3.  **Banglish**: Research findings ebong narration Bengali language e English script e bolben.

---

## Workflow

1.  **Multiple Searches**: Sudhu ekta search kore thamen na. Different wording use kore search koren.
2.  **WebFetch Context**: Search result theke relevant URL fetch koren for full documentation.
3.  **Synthesize findings**: Findings-gula punchy format e explain koren. AI slop (crucial, robust, etc.) avoid koren.

---

## Final Goal

Engineering logic define korar age research kore best existing pattern ba solution khuje dewa.

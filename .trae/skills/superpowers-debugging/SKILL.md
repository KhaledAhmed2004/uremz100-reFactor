# Superpowers Debugging Skill

Superpowers methodology follow kore systematic debugging ebong root-cause analysis korar jonno ei skill.

---

## CRITICAL: Debugging Philosophy

1.  **Systematic Approach**: Jump kore code change korben na. Prothome problem bujhen, hypothesis toiri koren.
2.  **Root-Cause Analysis**: "What actually happened?" ebong "Why it happened?" ber koren.
3.  **Banglish**: Hypothesis, analysis findings, ebong solution path Bengali language e English script e bolben.

---

## Debugging Workflow

1.  **Identify Issue**: Problem ta define koren. Input, expected output, ebong actual output check koren.
2.  **Hypothesize**: Kibhabe issue ta hoyeche sheta hypothesize koren.
3.  **Trace Root Cause**: Logic flow follow koren error trace korar jonno.
4.  **Verify Fix**: Solution apply koren ebong issue ta fixed kina verify koren.

---

## Final Goal

Logic level theke bugs remove kora ebong robust solution dewa.
